> **Experience the weather, don't just read it.** Atmosphere transforms your weather forecast into a visual, living experience. Built entirely with Vanilla HTML, CSS, and JS.

## ✨ Features

- 🌌 **Live Animated Backgrounds:** Pure CSS & JS driven dynamic skies with moving clouds, rain particles, snow flakes, and interactive lightning flashes that adapt speed to weather intensity.
- 💧 **Interactive Particles:** Custom CSS animations overlay the backgrounds to simulate falling rain, drifting snow, or twinkling stars.
- 📱 **iOS Glassmorphism UI:** A sleek, frosted-glass design language inspired by the highest-end mobile operating systems.
- 🗺️ **Live Interactive Radar Map:** Powered by Windy, watch real-time precipitation and storm clouds move directly on an interactive embedded map.
- 📅 **Expandable Forecasts:** View the upcoming 7 days. Click on any specific day to instantly slide out its detailed 24-hour timeline.
- 🔍 **Intelligent Autocomplete:** Debounced real-time city searching using the Open-Meteo Geocoding API.
- ⚡ **Zero Dependencies:** Lightning fast performance with no external frontend frameworks like React or Vue.

## 💻 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sxlmwn/atmosphere-weather-app.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd atmosphere-weather-app
   ```
3. **Run a local server:**
   You can use VSCode Live Server, or Python's built in HTTP server:
   ```bash
   python -m http.server 8080
   ```
4. **Open in Browser:**
   Navigate to `http://localhost:8080`

<br />

## 🎨 Interactive Elements

- **Click a Day:** Click any day in the 7-day forecast to expand its detailed 24-hour timeline.
- **Click a Detail Card:** Click the bottom *Wind*, *Humidity*, or *Feels Like* cards to spawn a frosted glass modal containing deep atmospheric insights like pressure and wind gusts.
